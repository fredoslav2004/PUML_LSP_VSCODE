# PlantUML LSP Extension

A modern VS Code extension for PlantUML support using the Language Server Protocol.

## Features

- **LSP Support**: Fast and efficient language features.
- **Diagnostics**: Real-time feedback on missing `@startuml` or `@enduml` tags.
- **Live Preview**: Visual feedback with a side-by-side preview panel (Powered by PlantUML.com).
- **IntelliSense**: Completions for common PUML keywords and diagram types.
- **Snippets**: Quick start templates for Sequence, Class, and Mindmap diagrams.
- **Syntax Highlighting**: Modern TMLanguage grammar for PUML.

## Usage

1. Open a `.puml` or `.plantuml` file.
2. Use the **PlantUML: Open Preview** command (icon in the editor title bar) to see your diagram.
3. Start typing keywords like `participant`, `actor`, or `class` to see completions.
4. Try typing `puml:sequence` for a quick template.

## Requirements

- VS Code 1.75.0 or newer.
- Internet connection (for the live preview rendering via PlantUML.com).
